# AI Efficiency Audit Builder

Premium live audit builder for composing consultant-grade AI efficiency audits with an immediate report preview.

## Product focus

- Builder-first workflow with live editing and live preview
- Two-panel desktop experience: structured editor on the left, polished report preview on the right
- Responsive mobile/tablet workspace with builder/preview switching
- Modular report sections with toggling and reordering
- Deterministic scoring, recommendation generation, and ROI framing kept in the same audit document state
- Saved audit portfolio and comparison view retained as secondary workflows

## Stack

- Vite
- React
- TailwindCSS
- Strict TypeScript typecheck via `tsc --noEmit`

## Local run

```bash
npm install
npm run dev
```

## Checks

```bash
npm run typecheck
npm run build
```
